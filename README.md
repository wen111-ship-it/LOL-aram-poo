Bonjour Mesdames, Messieurs,
Ce projet, entièrement réalisé en C#, vise à développer les capacités de l’élève dans ce langage.
Ce programme permet de simuler le célèbre jeu de bataille navale directement dans le terminal de votre éditeur de code.
Il a été conçu en respectant un cahier des charges disponible dans le répertoire du projet.
